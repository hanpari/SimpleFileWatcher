# SimpleFileWatcher

[![Build & Test](https://github.com/hanpari/SimpleFileWatcher/actions/workflows/tests.yml/badge.svg)](https://github.com/hanpari/SimpleFileWatcher/actions/workflows/tests.yml)

A lightweight and easy-to-use file watcher library for .NET.  
Monitors file system changes and triggers callbacks when files are created, modified, or deleted.

---

## ⚠️ Caution

This is a simple hobby project currently **in progress**.
  
APIs may change, features may be incomplete, and stability is not guaranteed.

---

## 🚀 Features

- Simple API for watching directories
- Watches create / change / delete events
- Minimal dependencies
- Works on Windows, Linux, and macOS
